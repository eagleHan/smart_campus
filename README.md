# smart_campus

智慧校园项目 — 基础设施与文档仓库

## 子仓库

| 仓库 | 说明 |
|---|---|
| [smart-campus-server](https://github.com/eagleHan/smart-campus-server) | 后端（基于芋道 ruoyi-vue-pro） |
| [smart-campus-web](https://github.com/eagleHan/smart-campus-web) | 管理台前端（基于 yudao-ui-admin-vue3） |
| [smart-campus-app](https://github.com/eagleHan/smart-campus-app) | 移动端 UniApp |

## 本地开发启动中间件

```bash
cd smart-campus-infra
docker compose -f docker-compose.dev.yml up -d
```
